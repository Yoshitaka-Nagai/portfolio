ABOUT

永井　善孝

Yoshitaka-Nagai

北海道札幌市出身、大学生から函館へ、前職は病院の事務員として勤めていました。
趣味はランニング。健康維持の為、週に2日程5kmをゆっくりと走っています。

現在は職業訓練校にて、WEBデザイン、HTML/CSS、JavaScriptを学び、フロントエンドエンジニアを目指して日々勉強、作品制作をしています。
Vue.jsなどのフレームワークにも興味があり、訓練校で学ぶこと以外に独学で新しい技術を身に付けていけるよう努力しています。

SKILL

  • HTML LS / CSS3

    デザインカンプからコーディングが出来ます。主にBEM規則にてCSSを記述しています。scssによる記述、レスポンシブ対応のコーディングも可能です。

  • JavaScript

    Vanilla JS、jQueryを使用し基本的なDOM操作を行うことが出来ます。swiper.js、GSAPなどのライブラリを使用したアニメーション作成も可能です。

  • WordPress

    既存のテーマを使用したWEBサイト構築が可能です。また、既存テーマのCSSを編集してサイトのデザインを変更することも出来ます。

  • Photoshop

    写真の加工、モックアップ作成、ロゴの作成を行うことが出来ます。また、用途に応じた画像形式の選択、画像のファイルサイズ調整なども可能です。

  • XD / Figma

    ワイヤーフレーム、デザインカンプ作成を行うことが出来ます。ホバーアクション、リンク遷移などの簡単なプロトタイプ作成も可能です。

  • Git

    Gitを使用して個人でのバージョン管理を行うことが出来ます。クライアントソフトはGitHub DesktopとSourcetreeの使用経験があります。

  • Visual Studio Code

    HTML、CSS、JavaScriptのコーディングに使用しています。拡張機能を使用して、SCSSのコンパイルやローカルサーバーを立ち上げてwebサイトの確認を行っています。

  • Word / Excel

    Wordは、文章作成、画像や図形の挿入、印刷設定などを行うことが出来ます。Excelは、簡単な関数を使用した表の作成、グラフ作成、印刷設定などを行うことが出来ます。
